17 May 2023 / Microsoft Azure and GitHub Azure and GitHub 101: My Experience

Hi everyone! I'm so excited to share with you my experience of attending an introductory session on Microsoft Azure and GitHub. It was a great opportunity to learn more about these amazing tools and how they can help me as a computer science student. The session was organized by the Microsoft Student Ambassadors and it covered the basics of cloud computing, Azure services, GitHub repositories, and GitHub Actions. I learned how to create an Azure account, set up a GitHub repository, and deploy a web app using GitHub Actions. It was very easy and fun! I also got to interact with other students and mentors who were passionate about technology and innovation. They gave me some useful tips and resources to continue my learning journey. I'm looking forward to exploring more features of Azure and GitHub and applying them to my projects. If you are interested in learning more about Azure and GitHub, I highly recommend you to check out the Microsoft Learn platform and the GitHub Learning Lab. They have some awesome courses and tutorials that will help you get started. Thank you for reading my blog post. I hope you enjoyed it and learned something new. Stay tuned for more updates on my adventures in computer science!
